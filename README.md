Recipe Manager
Python-проект для управления рецептами и списками покупок.

Проект реализует систему классов для работы с ингредиентами, рецептами, диетическими рецептами и итоговым списком покупок. Программа позволяет создавать ингредиенты, объединять их в рецепты, масштабировать количество порций и автоматически суммировать продукты из разных рецептов.
Реализованные классы

Ingredient

Класс для хранения информации об ингредиенте: название, количество и единица измерения.

Recipe

Класс для хранения рецепта и работы со списком ингредиентов.

DietaryRecipe

Наследник класса Recipe, который дополнительно хранит диетическую категорию рецепта.

ShoppingList

Класс для создания итогового списка покупок на основе одного или нескольких рецептов.
